Pour déployer le backend sur Railway et le web sur Vercel, utilisez les commandes suivantes :

```
# Déploiement backend sur Railway

npm run railway:deploy

# Déploiement web sur Vercel
npm run vercel:deploy
```

Ces commandes doivent être lancées à partir du répertoire racine du projet. Avant d'exécuter ces commandes, assurez-vous d'avoir résolu tous les problèmes d'authentification GitHub.
